# Motion-Glove-PSoC6-063-Code
The Motion Glove project utilizes a PSoC-6-063 to get sensor data from the BMI270 Gyroscope, and the voltage reading across the flex sensors. The flex sensors, which are on all five fingers, are used as a throttle input for the wheelchair. The BMI270 gives the current angle/heading (tbd) that steers the wheelchair
